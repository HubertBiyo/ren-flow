# ren-flow 路由 eval · 语料

> 这是「用户原话 → 期望路由到的 ren-* 技能」的测试语料。改了任何技能的 `description:` 或 `ren` 路由表后,按 `README.md` 的重放流程跑一遍,确认路由没退化。
>
> **范围**:只测 ren-* 体系内部路由。与体系外其它评审 / 工作流类技能的撞车不在此,由各自项目的约定管。
>
> 期望列取值:某个 `ren-*`(带模式时标注)/ `澄清`(应反问不硬路由)/ `让位`(外部系统接管)/ `直接改`(太小不走流程)。
